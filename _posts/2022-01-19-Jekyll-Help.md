# Jekyll Help

## Rebuild Jekyll Locally
To rebuild Jekyll locally without having to push to the repo and see the changes live, on a computer with Jekyll installed, you can run
> `bundle exec jekyll server`


This will rebuild and run a temporary server at localhost:4000 for you to view the changes
If you want to see live changes to this locally hosted version use the flag --livereload when launching.
